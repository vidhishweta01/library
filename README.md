
# library

![Screenshot](./assets/Screenshot1.png)

![Screenshot](./assets/Screenshot2.png)

A small library app.

Users can create books and mark those books as either Finished reading or Not yet read. Users can also delete the books if they like.

## Built With

javascript
HTML
CSS
Bootstrap
